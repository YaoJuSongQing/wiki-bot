# Wiki Q&A Bot - 好友同步方案

## 你这边（维护者）

1. 把项目推到自己 GitHub：
   cd ~/wiki-bot
   git init
   git add .
   git commit -m "init"
   git remote add origin git@github.com:你的用户名/wiki-bot.git
   git push -u origin main

2. 以后改了代码就：
   git add -A && git commit -m "更新了什么" && git push

## 朋友那边（接收者）

1. 第一次安装（只需一次）：
   git clone git@github.com:你的用户名/wiki-bot.git
   cd wiki-bot
   双击 安装.bat
   编辑 config.yaml 填自己的 API key
   双击 启动WikiBot.bat

2. 以后更新（你 push 后他运行）：
   双击 update.bat
   如果有新 wiki 数据文件，会自动加载
